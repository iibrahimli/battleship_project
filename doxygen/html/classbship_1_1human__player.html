Constructor with grid and game pointers.

Constructs a player with given grids and game. The grids are pointers to bs_grid objects inside the relevant game object. Players get all game state information through their grids. Players are free to keep track of any additional data they deem useful. The game is a pointer to the game on which the player makes moves
